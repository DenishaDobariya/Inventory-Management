# Inventory Management

## Installation & Setup

**1.Go to directory**

    cd Inventory-Management

**2.Setup backend server**

- Go to directory

      cd backend

- Install dependencies

      npm install

- craete .env file

      PORT=5000
      MONGODB_URI=your-mongodb-string
      JWT_SECRET= your-secret

- start the server

      npm run dev

**3.Setup frontend server**

- Go to directory

      cd frontend

- Install dependencies

      npm install

- start the server

      npm run dev
